# RNG
True Random Number Generator based on user mouse movement

Implementation based on article: Hu, Yue & Liao, Xiaofeng & Wong, Kwok-wo & Zhou, Qing. (2009). A true random number generator based on mouse movement and chaotic cryptography. Chaos, Solitons & Fractals. 40. 2286-2293. 10.1016/j.chaos.2007.10.022. 

Postprocessing algorithm based on: Hariyanto, Eko & Rahim, Robbi. (2016). Arnold's Cat Map Algorithm in Digital Image Encryption. International Journal of Science and Research (IJSR). 5. 6-391. 10.21275/ART20162488.
